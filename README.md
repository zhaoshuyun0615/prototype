# 产品原型

用于沉淀信息架构、流程、页面原型和交互说明。

## 建议文档

- `01_information_architecture.md`：信息架构
- `02_user_flows.md`：用户流程
- `03_page_inventory.md`：页面清单
- `04_interaction_notes.md`：交互说明
- `05_ui_design_guidelines.md`：高保真 UI 全局设计规范
- `06_online_feedback_prototype.md`：在线反馈功能一期原型说明
- `wireframes/`：低保真线框图
- `mockups/`：高保真设计稿或截图

## 原型交付硬性要求

- 后续所有 HTML 原型必须遵循 `00_global_constraints/04_constraints.md` 中的“原型设计硬性要求”。
- 高保真 UI 和 HTML 实现细则统一遵循 `03_prototype/05_ui_design_guidelines.md` 中的“HTML 原型交付硬性规范”。
- `index.html` 作为 iframe 平铺总入口；每个页面或状态单独拆分为独立 HTML 文件。
